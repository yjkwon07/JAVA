트랜잭션을 적용하지 않은 경우 (송금)

결과

	SQL Developer로 각 계좌 잔고를 조회.
	이번에는 트랜잭션이 적용되었으므로 김유신의 잔고는 물론이고 오류가 발생하지 않은 홍길동의 잔고도
	원래의 금액으로 롤백이 된다.

	대부분의 애프리케이션에서는 이처럼 Service 클래스에 트랜잭션을 적용.
	사실 SpringMybatis2에서 구현한 회원 기능 프로그램에서 MemberServiceImpl 클래스에도 트랜잭션 애너테이션을 적용했다.
